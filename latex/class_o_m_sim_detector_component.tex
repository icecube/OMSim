\doxysubsection{Detailed Description}
This class serves as a base class for constructing and manipulating detector components in \doxylink{class_o_m_sim}{OMSim}. It provides a set of helper functions for creating and arranging components within a detector, as well as methods for altering the geometry, such as subtracting one component from another. The class maintains an internal list of components, allowing for operations to be performed on all components simultaneously. The {\ttfamily place\+It} methods allow for the placement of components within a specified logical volume, either via individual positioning and rotation parameters, or via a Geant4 transformation. Furthermore, components from one {\ttfamily \doxylink{class_o_m_sim_detector_component}{OMSim\+Detector\+Component}} instance can be integrated into another, with user-\/specified position, rotation, and name extension. This makes it possible to build complex geometrical structures in a modular fashion in cases where a single mother volume cannot be used. \doxysubsubsection*{Classes}

This function is used to add a new component to the \textquotesingle{}Components\textquotesingle{} vector. Each component contains the solid volume, logical volume, position, rotation, and a unique name. It checks first if a component with the same name already exists. If it does, a warning message is generated and the name of the new component is appended with a unique suffix to avoid conflicts. The new component is then added to the \textquotesingle{}Components\textquotesingle{} vector.

The provided position and rotation parameters are used to transform the components from the input \doxylink{class_o_m_sim_detector_component}{OMSim\+Detector\+Component} instance before integrating. Each transformed component from the input instance is then appended to the current instance using the \doxylink{class_o_m_sim_detector_component_af079cb221a23adc315b1b0b3dc55c814}{append\+Component()} function. A name extension is applied to the original component names for distinction after integration.

The method goes through each component of the \doxylink{class_o_m_sim_detector_component}{OMSim\+Detector\+Component}, computes the transformation needed, and subtracts it from the input volume.
