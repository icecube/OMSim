◆ checkAbsorbedPhotons()

G4bool OMSimSensitiveDetector::checkAbsorbedPhotons ( G4Step *  aStep)
private

This function checks if the particle in the given step is an optical photon and if it's either the last step in the volume or if the post step process was absorption. If either of these conditions is met, the photon is considered absorbed and the function returns true.

Parameters
aStepThe step in which the particle is being checked.
Returns
True if the photon is absorbed, false otherwise.
